# aspire-test

## Build Setup

```bash
# install dependencies
$ npm install

# serve with hot reload at localhost:3000
$ npm run dev

# unit test cases
$ npm run test

# build for production and launch server
$ npm run build
$ npm run start

# generate static project
$ npm run generate
```
